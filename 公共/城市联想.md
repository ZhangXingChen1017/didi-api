# 城市联想

## 接口说明

根据输入信息联想城市名称

## 请求地址

`/v1/common/getAddress`

## 请求格式

`{url}?k1=v1&k2=v2&k3=v3&...`

## 请求方式

POST

## 请求参数

| 名称  | 类型   | 必选 | 描述               |
| ----- | ------ | ---- | ------------------ |
| city  | String | yes  | 城市名称，如深圳   |
| input | String | yes  | 搜索词，如深圳北站 |
