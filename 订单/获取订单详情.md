# 获取订单详情

## 接口说明

获取订单状态

## 请求地址

`/v1/order/detail`

## 请求格式

`{url}?k1=v1&k2=v2&k3=v3&...`

## 请求方式

POST

## 请求参数

| 名称           | 类型  | 必选 | 描述                     |
| -------------- | ----- | ---- | ------------------------ |
| order_id       | int   | yes  | 订单id                   |
| passenger_clat | float | no   | 订单实时经度（建议填写） |
| passenger_clng | float | no   | 订单实时纬度（建议填写） |

## 访问频率限制

- 订单接口访问增加频率限制
- 司机计费前（status<500）访问限制在 10秒/次
- 司机计费后（status>=500）访问限制在 300秒/次
- 如果超频：返回 errno: 36003
