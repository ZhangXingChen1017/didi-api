# 投诉选项接口

## 接口说明

该接口会根据指定订单的不同场景返回不同的投诉列表, 投诉前请调用此接口

## 请求地址

`/v1/complaint/getReasonList`

## 请求格式

`{url}?k1=v1&k2=v2&k3=v3&...`

## 请求方式

POST

## 请求参数

| 名称     | 类型   | 必选 | 描述                                             |
| -------- | ------ | ---- | ------------------------------------------------ |
| order_id | String | yes  | 要投诉的订单id(会根据订单的不同场景返回原因列表) |
